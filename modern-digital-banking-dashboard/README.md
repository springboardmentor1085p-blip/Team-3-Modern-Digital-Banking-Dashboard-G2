# Modern Digital Banking Dashboard

A full-stack banking dashboard application with React frontend and FastAPI backend.

## Features
- User authentication (JWT)
- Account management
- Transaction tracking
- Responsive dashboard

## Prerequisites
- Docker & Docker Compose
- Node.js 18+ (for local development)
- Python 3.11+ (for local development)

## Quick Start

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd modern-digital-banking-dashboard

Commands for Execution:

1. cd modern-digital-banking-dashboard/modern-digital-banking-dashboard/backend
2. python -m venv venv
3. venv\Scripts\activate
4. uvicorn app.main:app --reload
5. cd modern-digital-banking-dashboard/modern-digital-banking-dashboard/frontend
6. npm install
7. npm start
